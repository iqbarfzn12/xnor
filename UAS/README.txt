Website ini merupakan personal website atau  portofolio sederhana, Tujuan dari website ini adalah untuk memperkenalkan diri saya  dan membagikan informasi tentang pengalaman kerja, kemampuan yang dimiliki, serta memberikan kontak untuk dapat dihubungi.
Website ini berisi beberapa section, antara lain :
1.	Navbar: Berisi menu navigasi yang berguna  untuk melompat ke bagian yang ingin dituju.
2.	Header: Menampilkan gambar profil dan paragraf singkat yang memperkenalkan diri saya.
3.	Experience: Bagian ini menampilkan beberapa pengalaman kerja saya.
4.	Skill: Menampilkan daftar kemampuan yang dimiliki oleh saya. Setiap kemampuan disertai dengan indikator persentase.
5.	Contact: Bagian ini menyediakan informasi kontak seperti alamat, email, nomor telepon, dan tautan ke profil LinkedIn saya.
6.	Footer: Pada section terakhir ini hanya menampilkan hak cipta dan nama pemilik website.

Komponen yang ada di dalam website :
1.	HTML, sebagai kerangka websitenya
2.	CSS, disini saya menggunakan external css agar lebih mudah diatur
3.	Bootsrap, sebagai CSS framework, agar lebih mudah dan rapih
4.	Boxicon, sumber semua icon-icon yang ada di website saya
